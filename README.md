# Lawrence Jablonski

I'm a Technical Support Engineer at Consensys working on MetaMask, where I help troubleshoot complex Web3 issues and bridge the gap between users, support, and engineering.

I build practical tools at the intersection of Web3 support, automation, and data analysis that make blockchain data easier to understand. 

My projects include blockchain explorers, transaction trackers, support utilities, and workflow tools designed to help teams investigate issues more efficiently.


---

## 🛠️ Projects

| Project | Description |
|--------|-------------|
| [DoraScan](https://github.com/Labronski/dorascan) | Multi-chain blockchain explorer: Find wallet addresses and transaction hashes across networks from one interface. |
| [Bridge Aggregator](https://github.com/Labronski/bridge-aggregator) | Aggregates cross-chain bridge transaction data across providers into a single normalized investigation view. |
| [Perpetuals Tracker](https://github.com/Labronski/perpetuals-tracker) | Tracks leveraged trading activity by wallet address via the Hyperliquid API. |
| [Predictions Tracker](https://github.com/Labronski/predictions-tracker) | Tracks prediction market positions and trade history by wallet address via the Polymarket API. |
| [State Log Explorer](https://github.com/Labronski/state-log-explorer) | Parses raw JSON application state logs into a structured, readable format for debugging. |
| [Support Conversation Exporter](https://github.com/Labronski/support-conversation-exporter) | Flask app that queries the Intercom API to export support conversations to CSV by date range. |
| [AI Support Training Generator](https://github.com/Labronski/ai-support-training-generator) | Generates training materials for support agents automatically using Intercom conversation data and GPT-4. |
